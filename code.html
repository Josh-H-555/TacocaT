TacocaT

TacocaT is a project that will check if a word is palindrome.

getValues
getValues will receive the input from the form, and call the methods to check whether the input is a palindrome, and display whether it is or not.

checkPalindrome
To check for a palindrome, we need to compare whether the letters at the opposite end of the word are the same, and continue to check if letters are the same moving inward. Intuitively, we only need to check half of the string, as if we subtract 'i - 1', we will get the corresponding value index to the opposite letter. We use the Math.floor function to give us a whole number the event the word has an odd number of letters. If the number of letters is odd, we don't need to check the middle letter, as the indices will hone in on that single letter. If we find a mismatch, we immediately return false for more efficiency.

displayPalindrome
Will check if the word is a palindrome. If it is, the alert box will be green, if it's not, the alert box will be red. Will print a message corresponding to whether it is true or false.
